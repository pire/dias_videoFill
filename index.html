DIAS video container filler Github repo

A simple JS snippet to make a <video> or <iframe> fill it's container. Works with responsive implementations too.

Dependencies

  • jQuery

Instructions

  1. load the dias_videoFill.js and dias_videoFill.css into your site
  2. insert each <video> or <iframe> in a container .dias-video-filler

Browser compatibility

  • Chrome
  • Firefox
  • Safari
  • IE 7+
