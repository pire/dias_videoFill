DIAS video container filler Github repo

A simple JS snippet to make a <video> fill it's container. Works with responsive implementations too.

Dependencies

  • jQuery

Instructions

  1. load the dias_videoFill.js and dias_videoFill.css into your site
  2. insert each <video> in a container .dias-video-filler

Browser compatibility

  • Chrome
  • Firefox
  • Safari
  • Opera
  • IE 9+
